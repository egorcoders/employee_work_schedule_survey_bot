# Бот опросников
Ежедневно опрашивает сотрудников о режиме работы 

### Опросы отправляются на адрес:
SEND_POLL 

### Для получения дополнительных данных:
GET_UPDATES

### Для работы бота, необходим токен и ID чата:
TOKEN 

CHAT_ID

### Периодичность опросов и время проведения:
POLL_DELAY

POLL_TIME
